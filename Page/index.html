Welcome to KToast.

KToast for iOS - is an Android like Toast control developed for iOS 5+ and published under MIT license. We are maintaining this control using GitHub and you can fork it for development here.

$ cd your_repo_root/repo_name
$ git clone https://github.com/krupalghorpade/KToast.git

If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.

System Requirements!

This control is developed with Xcode 5 and compatible with iOS 5.0, tested on iOS 6 and iOS 7. It also requires a CoreGraphics framework to be linked with your project.
